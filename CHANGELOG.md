# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-20

### Added
- Initial release of Improved Editor Tralalero
- BlockHub popup with categorized blocks
- Search functionality
- Favorites system
- Scrollable grid
- Support for multiple categories (blocks, triggers, decorations)
- GitHub Actions CI/CD for automatic builds
- Comprehensive README and documentation

### Features
- Organized block selection in editor
- Real-time search
- Favorite blocks management
- Modern UI with scroll and buttons

### Technical
- Hook to EditorUI::onSelectCategory
- Custom FLAlertLayer popup
- Persistent favorites storage (placeholder)
- CMake build system
- Android64 and Android32 support
